Fluo is an authentication SDK for Flutter that lets you integrate complete UI flows (email, mobile, Google, Apple) in minutes across Android, iOS, Web, and macOS.

### Highlights

- Ready-made UI and flows with [FluoOnboarding](/quickstart)
- Works with your backend: [Firebase](/firebase), [Supabase](/supabase) or [Custom backend (JWT)](/custom-backend)
- Customizable theme with [FluoTheme](/theming)
- Support for 17 languages

### Get started

- Follow Installation, then Quickstart for a full example.
- Integrate your backend using the _Backend integrations_ section.
