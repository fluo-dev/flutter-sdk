Fluo is an authentication SDK for Flutter that lets you integrate complete UI flows (email, mobile, Google, Apple) in minutes across Android, iOS, Web, and macOS.

### Highlights

- Ready-made UI and flows: `FluoOnboarding`
- Works with your backend: Firebase, Supabase, or Custom (JWT)
- Built-in internationalization via `FluoLocalizations`

### Get started

- Follow Installation, then Quickstart for a full example.
- Integrate your backend via Guides.
