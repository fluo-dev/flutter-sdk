---
title: Installation
description: Install Fluo and set up your Flutter app.
---

### Requirements

- Flutter: >=3.8.0

### Dependency

```bash
flutter pub add fluo
```

### Flutter configuration

Add `FluoLocalizations.delegate` to your `MaterialApp` `localizationsDelegates`.

```dart
MaterialApp(
  // ...
  localizationsDelegates: const [
    FluoLocalizations.delegate,
    // ...other delegates...
  ],
)
```

### macOS (network)

Add these keys to:

- `{app}/macos/Runner/DebugProfile.entitlements`
- `{app}/macos/Runner/Release.entitlements`:

```xml
<dict>
  <!-- other keys -->
  <key>com.apple.security.network.client</key>
  <true/>
  <key>keychain-access-groups</key>
  <array>
    <string>$(AppIdentifierPrefix)$(CFBundleIdentifier)</string>
  </array>
</dict>
```
