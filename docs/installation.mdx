---
title: Installation
description: Install Fluo and set up your Flutter app.
---

### Requirements

- Flutter: >=3.8.0

### Dependency

```bash
flutter pub add fluo
```

### Flutter configuration

Add `FluoLocalizations.delegate` to your `MaterialApp` `localizationsDelegates`.

```dart
MaterialApp(
  // ...
  localizationsDelegates: const [
    FluoLocalizations.delegate,
    // ...other delegates...
  ],
)
```

### macOS (network)

Add this key to `{app}/macos/Runner/DebugProfile.entitlements` and `{app}/macos/Runner/Release.entitlements`:

```xml
<dict>
  <key>com.apple.security.network.client</key>
  <true/>
  <!-- other keys -->

</dict>
```
