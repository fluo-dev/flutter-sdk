### Usage

```dart
FluoOnboarding(
  fluoTheme: FluoTheme.native(),
  onUserReady: () => setState(() {}),
)
```

### Key properties

- `fluoTheme`: UI theme.
- `onUserReady`: callback when the user becomes ready.

### Notes

- Authentication and registration steps are handled automatically.
