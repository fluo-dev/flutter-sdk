### Configuration

Add `FluoLocalizations.delegate` in `MaterialApp` (see Installation).

### Supported languages

Based on `lib/l10n/*.arb`, Fluo includes: ar, de, en, es, fa, fr, hi, it, ja, ko, nl, pl, pt, pt_BR, ru, sv, tr, zh.

### Customization

If needed, you can override some texts by providing your own localizations above Fluo.
