---
title: Theming
description: Customize the look and feel with FluoTheme.
---

### Choose a base

- Native apps: `FluoTheme.native()`
- Web: `FluoTheme.web()`

```dart
FluoOnboarding(
  fluoTheme: FluoTheme.native(
    primaryColor: Colors.black,
    accentColor: Colors.white,
  ),
)
```
